# 《货币战争1：债务的魔咒》动画制作方案

## 项目概述

**项目名称**：货币战争动画系列 - 第一部：债务的魔咒
**原作**：宋鸿兵《货币战争1：债务的魔咒》
**动画类型**：教育性纪录片风格动画
**目标受众**：对金融、历史、经济感兴趣的观众
**总时长**：约60-90分钟（可拆分为多个短片）
**技术栈**：Remotion + React

---

## 一、章节规划与动画设计

### 总体结构：10集系列，每集6-9分钟

#### 第1集：序幕 - 货币的本质

**核心内容**：

- 什么是货币？财富的索取权
- 黄金与白银作为终极财富代表
- 纸币的出现：从收据到货币
- 1971年：黄金非货币化
- 债务货币的诞生

**动画场景设计**：

1. **开场**：金条在阳光下闪闪发光，逐渐过渡到古代集市
2. **货币演变**：金银硬币 → 纸币收据 → 现代法币
3. **时间线动画**：标注重要年份（1694年英格兰银行、1971年尼克松冲击）
4. **核心概念可视化**：
   - 金银金字塔堆叠
   - 国债像锁链一样缠绕货币
   - 通货膨胀吞噬财富的视觉隐喻

**视觉风格**：深色调，金色点缀，史诗感

**Remotion实现思路**：

```typescript
// 伪代码示例
const EuropeMap: React.FC = () => {
  return (
    <AbsoluteFill>
      <Img src={europeMap} />
      {/* 信使动画路径 */}
      <Sequence from={0} duration={120}>
        <Mpath
          path="M 100 200 L 150 180 L 200 220"
          stroke="gold"
          strokeWidth={3}
          strokeDasharray="1000"
          strokeDashoffset={interpolate(0, 1000)}
        />
      </Sequence>
    </AbsoluteFill>
  );
};
```

## 十、风险评估与应对

### 风险1：史实争议

**风险描述**：书中部分观点存在争议
**应对措施**：

- 在开头声明："本书为作者一家之言，请独立思考"
- 交叉验证关键史实
- 提供参考来源列表

### 风险2：技术复杂度

**风险描述**：某些场景（如战争场面）制作难度大
**应对措施**：

- 采用信息图表风格，避免写实战争场面
- 使用符号化、抽象化的视觉表达
- 重点放在金融逻辑而非历史细节

### 风险3：版权问题

**风险描述**：使用历史人物肖像可能涉及版权
**应对措施**：

- 使用风格化、剪影风格的人物形象
- 避免直接使用照片
- 所有插画原创绘制
