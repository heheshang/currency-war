## File Structure Observations

1. **Double Table of Contents**: Each book has its chapter list listed twice:
   - First in the main book introduction section (brief listing)
   - Second right before each chapter's actual content begins

2. **Content Separation**: There is clear separation between:
   - Introduction/Preface material
   - Chapter outline/table of contents
   - Actual chapter content

3. **Navigation Markers**: The file contains "返回总目录" (Return to Main Table of Contents) and "返回货币战争①" (Return to Currency Wars 1) navigation markers

4. **Special Formatting**: The text uses various formatting elements:
   - 〖...〗 for prominent quotes
   - 【...】 for section headings
   - Arrows and other decorative elements

5. **Book 4 Note**: Book 4 only has 9 chapters, unlike the first three books which each have 10 chapters
